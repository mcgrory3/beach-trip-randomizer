# Beach Trip Randomizer

**Version:** `db8a053`  
**Live:** https://mcgrory3.github.io/beach-trip-randomizer/

## Features
- Pre-spin setup: total budget, people, weekend timeline, travel style
- Weekend-only trips (back to work Monday)
- 4 commercial budget styles (never private, never first class)
- All-in cost: flights, parking, lodging, food, transport, activities
